## Functions
### Simple function
``` 
def square(x: Double) = x * x
```
### Function Inception
``` 
def sumOfSquare(x: Double, y: Double) = square(x) + square(y)
```
### If
``` 
def abs(x: Int) = if (x >= 0) x else -x
```
``` 
def or(x: Boolean, y: Boolean) = if (x) true else if (y) true else false
```
